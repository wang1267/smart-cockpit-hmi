# 🚗 模拟驾驶仪表盘

> 炫黑科技风 · 语音控制 · 电池管理系统  
> Python + Flask + Web 前端 | 零硬件依赖

## 运行

```bash
cd backend
pip install -r requirements.txt
python app.py
```

浏览器打开 `http://localhost:5000`

## 功能

| 模块 | 功能 | 操作方式 |
|------|------|----------|
| 🚀 仪表盘 | 数字车速 + RPM + 功率 + 档位 | 点 P/R/N/D 按钮 / 键盘 |
| 🎚 踏板 | 油门 / 刹车拖拽滑块 | 鼠标拖拽 / 触屏 |
| 🔋 电池 | SOC 百分比 + 续航 + 温度 + 瞬时功率 | 实时显示 |
| ⚠ 电量告警 | 20% · 10% · 2% 分级提醒 | 自动触发，每级仅一次 |
| ⚡ 充电 | 按钮或语音触发，满电自动停 | 点按钮 / 语音 |
| 💀 断电保护 | 0% 全系统冻结 | 遮罩内置充电按钮 |
| 🌡 空调 | 温度 ± · 风量 ± · 制冷/制热/AUTO | 点按钮 / 键盘 ↑↓ / 语音 |
| 🎤 语音唤醒 | 「小王小王」唤醒，8 秒无指令休眠 | 语音 / 点击浮标 |
| 🗣 定速巡航 | 自动维持目标车速 | 语音设定 |
| ⌨ 键盘 | P/R/N/D 换挡 · ↑↓ 调温 · ←→ 松踏板 | 键盘 |

## 语音指令

| 功能 | 说法示例 |
|------|----------|
| 🅿 挂挡 | 「挂D档」「停车」「倒车」「空挡」 |
| 🚗 定速巡航 | 「**定速六十**」「保持80」「开到200」 |
| ❌ 取消巡航 | 「取消定速」「关闭巡航」 |
| 🏎 加速 | 「加速」「快一点」「**油门加满**」 |
| 🛑 减速 | 「减速」「慢一点」「**踩刹车**」 |
| 🌡 空调 | 「空调24度」「制冷」「制热」「风量5档」 |
| ⚡ 充电 | 「充电」「停止充电」 |
| 🤖 辅助驾驶 | 「开启自动驾驶」「关闭自动驾驶」 |

> 💡 唤醒后 8 秒内可说多条指令；支持中文数字（六十→60）；口音自动容错；听不懂会提示示例

## 技术栈

| 层 | 技术 |
|----|------|
